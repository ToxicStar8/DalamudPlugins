## AutoHunt
使用多方插件，像脚本一样按部就班的执行，实现跨服、换图、切线、导航的操作，以达成接近全自动狩猎的目的。</br>
使用说明：
1. 安装`Sonar`、`TeleporterPlugin`、`Lifestream`、`Vnavmesh`。
2. 打开`Sonar`的S怪聊天框`Echo`的生成与击杀提示。
3. 在设置中勾选需要监听的`S怪`。
4. 切换到队列分页，勾选自动按照队列进行狩猎，即可在任何位置第一时间前往监听到的`S怪`的现场。
